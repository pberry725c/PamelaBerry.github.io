SQL Projects

To enhance SQL proficiency, I completed a progressive ladder challenge which began with foundational concepts, advancing to common table expressions (CTEs), while integrating moving frames, a crucial concept for analyzing data trends such as stock prices. To apply these skills to real world problems, we were tasked with examining data provided by UK companies and determining the saturation of the gender pay gap in recent years. The project included defining quantifiable metrics and presenting insightful findings via visualizations created in Power BI. This experience not only provided hands-on exposure to various SQL functions but also strengthened my problem-solving skills for effectively engaging with real-world databases.
